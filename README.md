# python-sudoku-game


python-sudoku-game with pyqt5 small project

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install PyQt5.

```bash
pip install PyQt5
```

## Usage

```python
import PyQt5
```
## play sudoku if get stuck press solve it will solved
